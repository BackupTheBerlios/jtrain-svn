3. Menü Editoren

 Wird ein Editor aufgerufen, wechselt JTrain in den Editiermodus. Jetzt können keine Befehle gesendet werden. Das Hauptfenster wird blau.

3.1 Der Schaltdekodereditor

3.1.1 Protokoll

Hier können Sie einstellen, ob ein Digitalprotokoll für alle Schaltdekoder voreingestellt werden soll. Falls ja, machen Sie Ihr Kreuzchen, wählen Sie das Protokoll und drücken bitte dann den 'Speichern'- Schalter. Sie brauchen dann nicht mehr bei jedem Schaltdekoder einzeln das Protokoll zu wählen.

3.1.2 Digitaladresse

Hier können Sie die Digitaladresse des Dekoders angeben. Zulässiger Wertebereich: 1..255.

3.1.3 Protokoll

Hier können Sie das Digital-Protokoll des Dekoders einstellen. Zulässige Werte:

  • M Märklin/Motorola-Format

  • N NMRA-DCC-Format

  • P Protocol by Server: Der Server bestimmt den Protokolltyp

3.1.4 Startstellung

Die Stellung, die nach dem Einschalten von JTrain voreingestellt wird.
Standard: rot für Signale, grün für Weichen
Achtung: Dieser Wert kann von einem zuständigen Block überschrieben werden!

3.1.5.Schaltzeit

Hier kann eine Schaltzeit für diesen Magnetartikel eingegeben werden. Dies wird jedoch nicht von allen Servern umgesetzt (ausprobieren)!

3.1.6. Typ

Eigentlich egal, wird zur Zeit nur zur Vorauswahl der Startstellung benutzt in diesem Editor

3.1.7 wird benutzt

Diese Schaltfläche wird ausgewertet von 'alle MA testen'. Ist sie aktiviert, wird in diesem Testprogramm der Dekoder betätigt.

3.1.9 Ende
