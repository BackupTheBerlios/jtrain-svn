3.3 Der Gleisbildeditor

3.3.1 Dialog: Gleisbildeditor-Auswahl

Hier wählen Sie, ob Sie ein vorhandenes Gleisbild öffnen oder ein neues erstellen.
Wählen Sie 'Vorhandenes Gleisbild', erscheint daraufhin ein Auswahldialog mit den gespeicherten Gleisbildern.
Wählen Sie 'Neues Gleisbild', können Sie ihre Auswahl unter mehreren vorgegebenen Formaten treffen. Mit der Option 'maximum' machen Sie aber auch nichts falsch, denn das Format passt auf Ihren Monitor und kann später im Blockeditor noch auf die optimale Grösse reduziert werden.

3.3.2 Bedienung

Die Bedienung erfolgt weitestgehend mit der Maus. Die Symbole sind (hoffentlich) selbsterklärend. Der kleine Kreis ist das Symbol für einen Rüchmeldekontakt.
  • Neues Element: das gewünschte Element einfach anklicken, ein neues Element wird links oben im Gitternetz bereitgestellt. Dort können die Elemente auch gestapelt werden.
  • Element plazieren: Mit der Maus anfassen und an die Wunschposition ziehen. Die Feinjustage übernimmt JTrain.
  • Schaltelement plazieren: Nach dem Loslassen der Maus werden Sie nach der Digitaladresse dieses Elements gefragt. Diese geben Sie jetzt bitte ein. Sie können mehreren Elemente dieselbe Adresse zuordnen (und so z.B. Schutzweichen schalten), aber: Achten Sie bitte darauf, die Ausgänge Ihrer Dekoder nicht zu überlasten!
  • Rückmeldekontakt plazieren: Nach dem Loslassen der Maus werden Sie nach der Nummer dieses Kontakts gefragt. JTrain nummeriert durchgehend von 1 an durch. Benutzen Sie also S88-Module und plazieren den 6. Kontakt des 3. Moduls, so ist seine Nummer = 2 * 16 + 6 = 38; sie können beliebig vielen Kontakten dieselbe Nummer zuordnen und so auch Gleisbesetztmelder optisch realisieren.
  • Adressen bzw. Nummern ändern: Das Element mit der rechten Maustaste anklicken, danach die neue Adresse/Nummer eingeben.
  • Elemente löschen: Mit der Maus das zu entfernende Element nach oben in die Elementeauswahl ziehen und loslassen.

3.3.3 Menü Datei

Als Zeichen für Dateinamen sind zur Zeit nur die Buchstaben a-z, A-Z erlaubt.
Sollten sonst noch Fragen auftauchen, bitte ich um Mitteilung.

3.3.4 Menü Symbole

Hier können Sie zwischen 2 verschiedenen Formaten zur Darstellung der Gleisbildelemente wählen. Wollen Sie von der grösseren Auflösung mit den kleinen Elementen auf die kleinere wechseln, werden Sie gewarnt, dass Elemente unter Umständen nicht mehr dargestellt werden.
